7 自定义指令 #

7.1 概述 #

除了核心功能默认内置的指令 (例如 v-model 和 v-show)，Vue 也允许注册自定义指令。注意，在 Vue 中，代码复用和抽象的主要形式是组件。然而，有的情况下，你仍然需要对普通 DOM 元素进行底层操作，这时候就会用到自定义指令。

vue 中的自定义指令分为两类，分别是：全局自定义指令 和 私有自定义指令
