1.2 语法 #

if (条件判断)

//单条件表达式
if(n){语句块}//n会被转成布尔类型

☞ if ( 条件表达式【布尔类型的结果】 ) { 
   	    逻辑代码...
   }else {
         逻辑代码...
   } 

//多条件表达式
☞  if ( 条件表达式 ) {
      逻辑代码...
	  }else if ( 条件表达式 ) {
      逻辑代码...
    }else {
      逻辑代码...  
    }

1.3 执行过程 #

1. 单条件条件判断代码执行过程：
	 ✔ 先进行表达式结果判断 
     ✔ 如果结果是true, 程序只会执行if中的语句，不会执行else中的语句
	 ✔ 如果结果是false， 程序只会执行else中的语句，不会执行if中的语句
