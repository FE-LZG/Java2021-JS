3.6 通过索引方式赋值 #

var  ary = [];
ary[0]=1;
ary[1]=2;
数组：[20,22,36,19,26,18]
下标：[0, 1, 2, 3, 4, 5]
取值：数组[下标]
赋值：数组[下标] = 值；
总结：
    1. 数组中索引值是从 0 开始的
    2. 通过索引的方式给数组赋值，要按照顺序个数设置
    3. 通过  数组名.length 可以获取到当前数组的长度
